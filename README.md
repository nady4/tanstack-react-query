# 🤖 Tanstack react-query
TypeScript web app that uses Tanstack Query for data fetching, TailwindCSS for styling, and features user search and pagination.

<br>

## Tech stack

💬 **Languages:**

- TypeScript

🛠 **Frameworks/Libraries:**

- React
- Tanstack Query
- TailwindCSS

🧰 **Tools:**

- Visual Studio Code
- Git
